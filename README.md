# svelte-template-component

A better and more complete version of the official [sveltejs/component-template](https://github.com/sveltejs/component-template).

## Content

- Typescript support (with [#29](https://github.com/sveltejs/component-template/issues/29) fixed)
- External Typescript files support (not just in svelte components!)
- Prettier formatting
- Full CI experience with pre-made scripts and Github workflows
